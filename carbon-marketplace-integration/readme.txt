A WordPress plugin that integrates with multiple carbon credit vendor APIs to provide a meta search engine for carbon offset projects.

== Description ==

Carbon Marketplace Integration transforms your WordPress website into a meta search engine for carbon offset projects. The plugin integrates with multiple vendor APIs (CNaught and Toucan) to provide real-time project listings, search functionality, and affiliate-style checkout processes.

Features:
* Search and filter carbon offset projects by location, type, and price
* Real-time pricing from multiple vendors
* Affiliate-style checkout integration
* Order tracking and retirement certificates
* Elementor widget support
* Admin dashboard for managing API credentials and settings

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/carbon-marketplace-integration` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Carbon Marketplace->Settings screen to configure your API credentials

== Frequently Asked Questions ==

= What vendors are supported? =

Currently, the plugin supports CNaught and Toucan APIs, with the ability to add more vendors in the future.

= Do I need API credentials? =

Yes, you'll need API credentials from the supported vendors to fetch project data and process transactions.
